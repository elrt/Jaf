# Jaf
*Minimal interpreted programming language implemented in Java.*

## Examples
**Factorial:**
```java
func factorial(n) {
    if (n == 0) 1 else n * factorial(n - 1)
}
factorial(5)
```
**Array:**
```java
arr = [1, 2, 3]
arr[0] = 10
len = length(arr)
```
